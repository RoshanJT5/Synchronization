What it does

Browser audio capture

The Chrome extension captures the current tab audio and keeps the stream local to your network after WebRTC negotiation.

Multiple mobile speakers

Multiple phones can join the same session code. The sender creates one peer connection per receiver.

QR and deep links

QR codes point to this website with the session data. The website opens the installed app with syncronization://connect.

Screen-off playback

Android uses a foreground media-playback service. iOS declares background audio mode for remote speaker playback.
